Featured — VectorShift (YC S23) Frontend Assessment — Feb 2026
No-Code AI Pipeline Builder

A drag-and-drop visual pipeline editor built for VectorShift. Users connect nodes to construct AI workflows and submit them for real-time DAG validation — demonstrating full-stack architecture, reusable component design, and graph algorithm implementation.

  • Engineered a BaseNode abstraction reducing new-node creation from 80+ lines to under 20 — all nodes driven by a config object with zero boilerplate
  • Built 9 node types (Input, Output, LLM, Text, Filter, Transform, Merge, Prompt, Webhook) on a single shared component with unique color schemes and handles
  • Real-time variable detection — regex extracts valid JS identifiers and auto-generates ReactFlow input handles as the user types; textarea auto-resizes with content
  • FastAPI backend with Kahn's BFS algorithm for DAG cycle detection, returning num_nodes, num_edges, is_dag via POST /pipelines/parse
  • Zustand state management, polished result modal, full CORS integration, graceful error handling
